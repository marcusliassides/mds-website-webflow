# Fonts

Drop the licensed `.woff2` files here, then upload them to Webflow → Site Settings → Fonts.

The reference Next.js site keeps these in `mdswebsite/app/fonts/`:

```
MattyMattelSans-Regular.woff2     400
MattyMattelSans-SemiBold.woff2    600
MattyMattelSans-Bold.woff2        700
SourceSerif4-Regular.woff2        400
SourceSerif4-SemiBold.woff2       600
SourceSerif4-Italic.woff2         400 italic
SourceSerif4-SemiBoldItalic.woff2 600 italic
JetBrainsMono-Regular.woff2       400
JetBrainsMono-Medium.woff2        500
JetBrainsMono-SemiBold.woff2      600
```

Matty Mattel Sans is licensed for Mattel use only. Source Serif 4 is OFL (see `SourceSerif4-OFL.txt`). JetBrains Mono is OFL (see `JetBrainsMono-OFL.txt`).

These files are not committed to the repo — copy them in manually before the first Webflow import.
