Accessibility

Mattel Digital Studios is committed to making digitalstudios.mattel.com usable for everyone. We aim to conform to WCAG 2.2 AA across the site.

What that means in practice:

  • Keyboard-operable navigation, with a visible focus ring on every interactive element.
  • Color contrast verified against AA thresholds for all text and meaningful UI states.
  • Reduced-motion support — every animation collapses when prefers-reduced-motion: reduce is set.
  • Semantic landmarks (header, main, nav, footer) and live regions for any auto-rotating content.
  • 44px minimum target size on touch surfaces.

Reporting an issue

If something on this site doesn't work for you, write to accessibility@digitalstudios.mattel.com. We respond within five business days and prioritise WCAG-blocking issues.

Privacy

This site uses minimal first-party analytics and does not run third-party advertising trackers. Cookie use is described in the cookie notice the first time you visit. For Mattel's full privacy policy, see corporate.mattel.com/privacy.
